# Henry Ngo — GitHub Pages Portfolio

This is a lightweight, static portfolio suitable for GitHub Pages.

## Quick Start
1. Create a repo named **USERNAME.github.io** (replace USERNAME with your GitHub username).
2. Upload these files to the repo root (keep the `assets` folder).
3. Commit & push. GitHub Pages will serve it at **https://USERNAME.github.io**.

## Customize
- Edit `index.html` (name, bio, links, publications, projects, teaching).
- Replace the CV link in `assets/script.js` or set the correct path.
- Add a `cv/Henry_Ngo_CV.pdf` file to enable the CV download.

## Optional
- Add a custom domain via repo Settings → Pages → Custom domain.
- Add analytics (Plausible/GA) by inserting their scripts in `index.html`.
